# Bitcoin Script

## 简介

Bitcoin script 是一种非图灵完备的脚本，可能甚至谈不上语言。Bitcoin script 使用操作码 opcode 来定义动作，使脚本支持有限的自由操作。

Reference: [Script](https://en.bitcoin.it/wiki/Script)
