# 预言机是什么？

预言机是一种用来连接真实世界数据和区块链内部数据的桥梁。预言机往往会规范一套 API 供区块链使用。

预言机的分类：
- Software oracles
- Hardware oracles
- inbound and outbound oracles
- Centralized and decentralized oracles
