# 区块链是什么

区块链是什么？你可以放心地确信，目前整个世界上还没有准确的定义。区块链这个词来源于哪儿？是谁创造的？没有人知道。起源于比特币吗？比特币并没有提出 Blockchain 这个词。

这一章节从基础的数据结构开始，把区块链最核心的技术形态描述出来。
